# Pizza Yolo Design System

---

# Purpose

This document defines the visual language of Pizza Yolo.

It is not intended to create rigid design rules.

Instead, it establishes a consistent design language while allowing creative freedom where it strengthens the brand.

Pizza Yolo should never feel corporate.

It should feel designed.

---

# Design Philosophy

The design language is inspired by the intersection of:

- Modern streetwear brands
- Editorial design
- American fast food
- Summer culture
- Gaming aesthetics
- Contemporary web design

The result should feel:

- Confident
- Bold
- Clean
- Playful
- Memorable
- Young
- Premium through simplicity

Not luxurious.

Not minimal for the sake of being minimal.

Not trendy just because something is trending.

---

# The Core Principle

Every component should answer one question:

"Does this make Pizza Yolo feel more recognizable?"

If not,

remove it.

# Layout Philosophy

Layouts should feel intentional.

Not mechanical.

Design should guide attention naturally.

Whitespace is one of the primary design tools.

Do not fill empty areas simply because space exists.

---

# Controlled Imperfection

Unlike banking websites or SaaS dashboards, restaurant branding benefits from controlled imperfection.

Perfect mathematical symmetry can feel sterile.

Pizza Yolo intentionally allows small visual tension to create energy.

Examples include:

- Headlines that slightly overlap imagery
- Cards with intentionally varied proportions
- Offset photography
- Slightly broken grid alignments
- Unexpected cropping
- Asymmetrical section compositions

These decisions should always feel intentional.

Never random.

Never messy.

Think editorial magazine layouts rather than engineering diagrams.

The goal is excitement without losing clarity.

---

# Grid System

Use a 12-column responsive grid as the structural foundation.

However:

The grid is a guide.

Not a prison.

Elements may intentionally extend outside the grid if it creates stronger visual hierarchy or emotion.

Breaking the grid is acceptable when it improves composition.

---

# Spacing Philosophy

Large spacing communicates confidence.

Avoid cramped layouts.

Use breathing room generously.

Sections should feel separated by atmosphere rather than borders.

Whitespace creates rhythm.

---

# Typography

## Display Font

Jersey 10

Purpose

- Headlines
- Hero text
- Campaign slogans
- Posters
- Menu titles
- Packaging
- Event branding

Characteristics

Large.

Confident.

Recognizable.

Never use for long paragraphs.

Never use at small sizes.

Allow oversized typography whenever possible.

Typography should often become a design element rather than only information.

---

## Body Font

Poppins

Purpose

Navigation

Descriptions

Buttons

Product information

SEO copy

Supporting information

Characteristics

Highly readable.

Modern.

Neutral.

The body font should disappear.

The display font should be remembered.

---

# Typography Rules

Short headlines.

Long whitespace.

Minimal copy.

Large scale.

Confident composition.

Avoid center-aligning large amounts of text.

Limit paragraph width for readability.

# Color System

## Primary

Purple — `#885BB8`

Brand ownership.

Used for:

Packaging

Buttons

Highlights

Hero elements

Background blocks

Interactive states

---

## Secondary

Ink — `#303030`

Depth.

Contrast.

Confidence.

Used for typography, navigation and dark sections. Not a pure black —
always this exact off-black.

---

## Neutral

Cream — `#FFFDFA`

Creates clarity.

Allows photography to breathe.

Provides visual rest.

Not pure white — always this exact off-white. Cream and Ink are used in
roughly a 70/30 balance across the site (cream dominant).

On any purple (`#885BB8` / `#9B67D1`) surface, Cream is the only color
used for text and icons — never Ink, never pure white.

---

## Accent

Light Purple — `#9B67D1`

Use for gradients and hover/lighter variations of the primary purple.

Avoid introducing additional colors.

The more disciplined the palette becomes,

the stronger the brand becomes.

# Brand Texture

The subtle purple texture used across packaging should become a recurring visual asset.

Use it selectively.

Packaging

Posters

Hero backgrounds

---

# Logo Usage

Always provide generous clear space.

Never distort.

Never stretch.

Never rotate.

Prefer monochrome versions when possible.

Minimal.

Graphic.

Bold.

---

# UI Philosophy

Remove before adding.

Every component must justify its existence.

Simple interfaces feel premium.

Complex interfaces feel cheap.

---

# Accessibility

High contrast.

Readable typography.

Large tap targets.

Responsive layouts.

Keyboard accessibility.

Fast loading.

Accessibility should never compromise branding.

Good branding includes accessibility.

---

# Responsive Design

Mobile-first.

Design should feel intentionally crafted for every screen size.

Never simply shrink desktop layouts.

Recompose sections where necessary.

---

# Consistency

Consistency does not mean repetition.

Repeat principles.

Not layouts.

Every page should feel different.

Every page should still feel unmistakably Pizza Yolo.

---

# The 80/20 Rule

80% of the design should be highly consistent.

20% can surprise the user.

That 20% is where personality lives.

Unexpected layouts.

Oversized headlines.

Creative image crops.

Playful interactions.

Editorial compositions.

These moments make the brand memorable.

---

# Final Design Test

Before approving any screen, ask:

Does this feel like another restaurant website?

If yes,

keep iterating.

Does this feel unmistakably Pizza Yolo?

If yes,

it's ready.
